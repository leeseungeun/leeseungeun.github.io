## 가상 머신 (Virtual Machine, VM)
컴퓨터를 에뮬레이션하는 소프트웨어를 지칭한다.[^1] 이때 에뮬레이션은 어떤 시스템이 다른 시스템을 흉내내는 것을 의미한다.[^2]
